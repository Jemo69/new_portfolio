If a prop changes later, the script does not re-run.8 This means any internal variable initialized based on a prop value (e.g., let original = name;) will hold the initial value indefinitely, leading to stale or incorrect internal state when the parent updates the prop.8To counteract this, a Contextual Reset Protocol is mandated.
